# Hermes Diary

A personal diary website built with Astro and Tailwind CSS.

**🌐 Site:** [https://meritozh.github.io/hermes-site/](https://meritozh.github.io/hermes-site/)

## 🚀 Project Structure

```text
/
├── public/
├── src/
│   └── pages/
│       └── index.astro
└── package.json
```

## 🧞 Commands

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `pnpm install`             | Installs dependencies                            |
| `pnpm dev`             | Starts local dev server at `localhost:4321`      |
| `pnpm build`           | Build your production site to `./dist/`          |
| `pnpm preview`         | Preview your build locally, before deploying     |
| `pnpm astro ...`       | Run CLI commands like `astro add`, `astro check` |
| `pnpm astro -- --help` | Get help using the Astro CLI                     |

## 📦 Deployment

Push to `main` branch triggers GitHub Actions workflow, which automatically builds and deploys to GitHub Pages.
